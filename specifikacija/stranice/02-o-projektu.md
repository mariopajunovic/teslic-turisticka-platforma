# 02 — O projektu

📌 **TS 4.2** (statična stranica) · **Ruta:** `/o-projektu` · **Pristup:** javno

## Svrha
Informativna stranica koja objašnjava svrhu i ciljeve platforme, ko stoji iza nje i kako se zajednica može uključiti. Gradi povjerenje i kontekst.

## Sadržaj i raspored
```
[ Naslov stranice + kratak uvodni opis ]
[ 1. Misija i svrha platforme ]
[ 2. Ciljevi projekta (lista/ikonice) ]
[ 3. Kome je namijenjena (posjetioci · biznisi · autori) ]
[ 4. Nosilac i partneri projekta (logotipi) ]
[ 5. Kako se uključiti — CTA ka „Pridruži se” ]
[ 6. Kontakt blok / link ka Kontaktu ]
```

## Sadržaj sekcija (📌 TS 2 — ciljevi projekta)
- **Misija/svrha**: centralno digitalno mjesto za promociju Teslića (lokalna ponuda + turizam + priče).
- **Ciljevi**: promocija lokalnih proizvoda/usluga/atrakcija; veća vidljivost domaćih biznisa; upoznavanje Teslića kroz autentične priče i multimediju; podrška profilima registrovanih subjekata; pregled ponude i lokacija putem mape; samostalno upravljanje sadržajem od strane naručioca.
- **Kome je namijenjena**: kratki blokovi za tri publike s pripadajućim CTA.
- **Nosilac i partneri**: tekst + galerija logotipa (isti set kao footer).

## Komponente
- WYSIWYG tekstualni blokovi (uredljivo iz admina), slike, lista ciljeva s ikonicama, traka logotipa partnera, CTA dugmad.

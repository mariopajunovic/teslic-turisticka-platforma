# Recept: 02 — O projektu

📄 Spec: [stranice/02-o-projektu.md](../../stranice/02-o-projektu.md) · Tokeni: [00](../00-design-tokens.md) · Komponente: [01](../01-komponente.md)

## Artboardi
- `OProjektu – Default – Desktop` (1440)
- `OProjektu – Default – Mobile` (375)

## Blokovi
1. `Header` + `Breadcrumb` (Početna / O projektu)
2. **Hero/split** — naslov H1 „O projektu” + uvodni paragraf + ilustracija/slika.
3. **Misija i svrha** — tekstualni blok (H2 + body).
4. **Ciljevi projekta** — `Grid` od ikona+tekst stavki (6 ciljeva iz spec-a).
5. **Kome je namijenjena** — 3 kartice: Posjetioci · Biznisi · Autori (svaka s kratkim CTA).
6. **Nosilac i partneri** — tekst + traka logotipa partnera.
7. **CTASekcija** — „Pridruži se” (`primary`).
8. `Footer`

## Sadržaj
Statičan tekst (uredljiv iz admina). Ciljevi prepisati iz spec-a (promocija lokalnih proizvoda/usluga/atrakcija, vidljivost biznisa, autentične priče, profili subjekata, mapa, samostalno upravljanje).

## Stanja
- Statična stranica; nema dinamičkih stanja osim učitavanja.

## Responsive
- Grid ciljeva 3→2→1; kartice publika stack na mobilnom.
